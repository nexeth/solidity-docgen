# @nexeth/solidity-docgen

## Getting Started

### Install bun

If you haven't already, install bun. Follow the directions [here](https://github.com/oven-sh/bun)

### Linting

To lint your code:

```
bun lint
```

### Tesing

To run tests:

```
bun test
```
